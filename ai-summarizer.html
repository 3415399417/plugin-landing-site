AI 智能总结

网页太长时，先让 AI 摘出核心。支持当前页面总结、选中文本右键总结、手动粘贴、三种总结模式和 Markdown 导出。

Chrome 商店待上线 查看使用支持

把长网页变成可读摘要。

  • 总结当前网页正文，减少广告、导航和侧栏干扰
  • 选中文字后右键总结，只处理你关心的段落
  • 支持关键点、简洁总结、详细总结三种输出
  • 支持中文/英文输出、复制和导出 Markdown
AI summarizer side panel

核心：文章解释了一个产品如何从工具功能走向稳定工作流。

关键点：用户需要清晰入口、可信隐私说明和可持续的结果质量。

下一步：把摘要导出到 Markdown，用于笔记、复盘或团队分享。

三种总结模式

关键点

3-6 条短句，适合快速判断文章是否值得细读。

简洁总结

2-3 段概括，适合新闻、博客和普通资料页。

详细总结

包含背景、核心观点、例证和结论，适合研究和复盘。

中英文输出

同一篇网页可以切换输出语言，方便跨语言阅读。

使用流程

设置 DeepSeek API Key第一次使用时在设置页填写 API Key。
选择总结范围总结整页、选中文本，或手动粘贴内容。
选择模式和语言按阅读目的选择关键点、简洁或详细总结。
复制或导出把结果复制到笔记工具，或导出 Markdown 文件。
