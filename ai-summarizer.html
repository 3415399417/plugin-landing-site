AI Summarizer

简阅

AI 智能总结

把长网页、选中文本和粘贴内容变成清晰要点。免费每天 10 次，API Key 可选。

核心功能

网页总结

自动提取页面正文，适合新闻、资料页、技术文章和长文阅读。

选中文本总结

只处理你选中的段落，适合精读、查资料和局部摘录。

粘贴总结

打开侧边栏后直接粘贴文本，快速生成简洁总结或关键点。

Markdown 导出

总结结果可复制，也可以导出为 Markdown，方便放入笔记工具。

使用流程

安装后直接使用免费每天 10 次，无需先填写 API Key。
选择总结范围总结整页、选中文本，或手动粘贴内容。
选择模式和语言按阅读目的选择关键点、简洁总结或详细总结。
可选填写 API Key高级用户可填写自己的 DeepSeek API Key，使用个人额度。
