Privacy

隐私说明

我们的原则很简单：只在你主动触发功能时处理内容，不做广告追踪，不收集浏览历史。

会发送哪些数据？

当你使用总结或分析功能时，插件会把你选择处理的文本发送到 DeepSeek API。例如网页正文、选中文本、手动粘贴内容、小红书笔记文本、搜索结果标题，或截图 OCR 识别后的文字。

截图 OCR 会发送什么？

红势的图文识别功能只会在你主动上传、拖拽或粘贴截图，并点击识别时，把该图片发送到 OCR.space 做文字识别。识别结果会先显示给你确认或修改，再进入 AI 分析。

API Key 如何保存？

DeepSeek API Key 保存在 Chrome 扩展本地存储中，用于向 DeepSeek 发起请求。它不会发送到我们自有服务器。

本地会保存什么？

  • DeepSeek API Key
  • OCR.space API Key（如果你填写）
  • 语言偏好或使用次数
  • 短期缓存，用于减少重复请求
  • 你主动加入的小红书分析篮内容

不会做什么？

  • 不售卖用户数据
  • 不做广告追踪
  • 不持续监控网页
  • 不收集完整浏览历史

如何删除数据？

你可以在插件设置页清空 API Key，在插件界面清空分析篮，或直接从 Chrome 删除扩展来移除本地数据。
