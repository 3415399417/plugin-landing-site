Privacy

隐私说明

我们的原则很简单：只在你主动触发功能时处理必要内容，不做广告追踪，不出售用户数据，不持续监控网页。

红势会处理哪些数据？

当你主动使用红势的分析、OCR 或带题复刻功能时，插件可能会处理你主动提交的小红书笔记文本、当前页面中提取到的标题或正文、搜索结果标题、分析篮内容、截图 OCR 识别后的文字，以及你在带题复刻中输入的选题、产品、经历或观点。

AI 分析请求

未填写 DeepSeek API Key 时，红势每天提供 5 次免费 AI 分析。你主动提交的文本内容会发送到官方免费分析服务，再转发给 DeepSeek，用于生成分析结果。

如果你在设置页填写自己的 DeepSeek API Key，AI 分析请求会从浏览器直接发送到 DeepSeek，使用你的个人 API 额度，并不消耗每日免费次数。

截图 OCR 请求

当你主动上传、拖拽或粘贴截图，并点击识别图片文字时，图片会发送到 OCR.space 进行文字识别。OCR.space API Key 是可选项。

本地会保存什么？

  • 每日免费使用次数
  • 分析篮内容
  • 短期缓存结果
  • 可选的 DeepSeek API Key
  • 可选的 OCR.space API Key
  • 插件界面设置或语言偏好

我们不会做什么？

  • 不出售用户数据
  • 不做广告追踪
  • 不收集完整浏览历史
  • 不在用户未触发功能时持续监控网页
  • 未经用户操作，不会自动上传网页内容
  • 不会将用户 API Key 用于用户请求之外的其他目的

第三方服务

红势可能会使用 DeepSeek 生成 AI 分析结果，使用 OCR.space 识别截图中的文字，并使用官方免费分析服务为未填写 API Key 的用户提供每日免费分析额度。

关于分析结果

红势的 AI 分析结果仅基于用户主动提交或当前可见的文本内容生成，不包含账号权重、投流情况、发布时间、真实评论反馈等外部因素。分析结果仅供内容创作参考，不承诺一定涨粉、一定获得流量或一定产生商业结果。

如何删除数据？

你可以在插件设置页清空 API Key，在插件界面清空分析篮，清除浏览器扩展数据，或直接从 Chrome 扩展管理页删除红势插件。
