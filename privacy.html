Privacy

隐私说明

我们的原则很简单：只在你主动触发功能时处理内容，不做广告追踪，不出售用户数据，不持续监控网页。

会发送哪些数据？

当你使用总结或分析功能时，插件会处理你主动选择的内容，例如网页正文、选中文本、手动粘贴内容、小红书笔记文本、搜索结果标题，或截图 OCR 识别后的文字。

简阅 - AI 智能总结

简阅提供每天 10 次免费总结。未填写 API Key 时，你选择总结的文本会发送到官方免费总结服务，再转发给 DeepSeek 生成总结结果。

如果你在设置页填写自己的 DeepSeek API Key，请求会从浏览器直接发送到 DeepSeek，使用你的个人 API 额度。

红势 - 小红书爆款分析

红势会在你主动分析笔记、粘贴内容或上传截图时处理相关文本。图文识别功能只会在你主动上传、拖拽或粘贴截图并点击识别时，将图片发送到 OCR.space 进行文字识别。

API Key 如何保存？

你填写的 DeepSeek API Key 或 OCR.space API Key 会保存在 Chrome 扩展本地存储中。它们用于发起对应服务请求，不会被出售或用于广告追踪。

本地会保存什么？

  • 可选的 DeepSeek API Key
  • 可选的 OCR.space API Key
  • 每日使用次数
  • 语言偏好或界面设置
  • 短期缓存，用于减少重复请求
  • 你主动加入的分析篮内容

不会做什么？

  • 不出售用户数据
  • 不做广告追踪
  • 不持续监控网页
  • 不收集完整浏览历史
  • 不在你未触发功能时读取页面内容

如何删除数据？

你可以在插件设置页清空 API Key，在插件界面清空分析篮，或直接从 Chrome 删除扩展来移除本地数据。
