不是只给“感觉”，而是拆出可复用结构。

  • 单篇笔记：输出评分、结构拆解、爆款因子、复刻建议和改写模板
  • 搜索结果：批量提取标题和可见互动数，总结标题规律
  • 分析篮：把多篇笔记放在一起，找共性和可复制套路
  • 图文识别：上传、拖拽或粘贴截图，识别图片里的文字后分析
  • 手动粘贴：页面自动提取失败时也能继续分析

适合这些场景

把日常刷到的爆款笔记变成选题、标题和结构素材库。

新手博主

不知道爆文为什么火时，用它拆解标题、开头、收藏触发点，并判断值不值得复刻。

内容运营

批量复盘搜索结果，快速整理同赛道标题表达方式。

商家账号

参考竞品笔记的卖点组织、信任建立和评论引导方式。

代运营团队

把分析篮结果作为选题会和内容复盘的辅助材料。

使用流程

设置 DeepSeek API Key第一次使用时在插件设置页填写 API Key。
打开小红书页面支持笔记页、搜索结果页、手动粘贴文本和截图 OCR。
开始分析从侧边栏查看结构、评分、复刻建议、爆款因子和模板。
加入分析篮多篇笔记一起拆，输出共性规律和可复制套路。
