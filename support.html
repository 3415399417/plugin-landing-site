使用支持

这里整理两个插件的上手流程、API Key 说明和常见问题。

第一次使用

  1. 安装 Chrome 插件。
  2. 打开目标网页，点击插件图标或右键菜单。
  3. 简阅可直接使用每天 10 次免费额度。
  4. 红势目前需要在设置页填写 DeepSeek API Key 后使用。

简阅是否必须填写 API Key？

不必须。简阅默认每天可免费总结 10 次。高级用户可以在设置页填写自己的 DeepSeek API Key，填写后将使用个人 API 额度。

红势为什么需要 API Key？

红势是内容分析工具，当前版本使用你自己的 DeepSeek API Key 发起 AI 分析请求。这样成本由你自己掌握，插件不会托管你的 Key。

网页自动提取失败怎么办？

部分网页会限制内容提取。你可以选中文本后右键总结，或者复制内容粘贴到插件侧边栏里处理。

小红书图片文字读取不到怎么办？

红势 Pro 测试版支持截图 OCR。你可以上传、拖拽或粘贴截图，让插件先识别图片文字，再进入爆款分析。

如何删除本地数据？

在插件设置页清空 API Key，或从 Chrome 扩展管理页删除插件，即可移除本地存储的数据。
