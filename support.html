使用支持

这里整理两个插件的上手流程、免费额度、API Key 说明和常见问题。

第一次使用

  1. 安装 Chrome 插件。
  2. 打开目标网页，点击插件图标或使用右键菜单。
  3. 简阅每天可免费总结 10 次。
  4. 红势每天可免费分析 5 次。
  5. 超过免费额度后，可以第二天继续使用，或在设置页填写自己的 DeepSeek API Key。

简阅是否必须填写 API Key？

不必须。简阅默认每天可免费总结 10 次。高级用户可以在设置页填写自己的 DeepSeek API Key，填写后将使用个人 API 额度。

红势是否必须填写 API Key？

不必须。红势新版默认每天可免费分析 5 次，适合新用户先体验。超过次数后，用户可以等待第二天额度恢复，或填写自己的 DeepSeek API Key 继续使用。

为什么还保留 API Key 设置？

API Key 适合高频用户、团队测试或希望使用个人额度的人。填写后请求会从浏览器使用个人 Key 发起，不再消耗每日免费额度。

网页自动提取失败怎么办？

部分网页会限制内容提取。你可以选中文字后右键总结，或复制内容粘贴到插件侧边栏里处理。

小红书图片文字读取不到怎么办？

红势支持截图 OCR。你可以上传、拖拽或粘贴截图，让插件先识别图片文字，再进入爆款分析。

带题复刻怎么用？

先分析一篇爆款笔记，再输入你的选题、产品、经历或观点。插件会结合爆款结构，输出复刻匹配度、标题、开头、正文框架和风险提醒。

如何删除本地数据？

在插件设置页清空 API Key，清空分析篮，或从 Chrome 扩展管理页删除插件，即可移除本地存储的数据。
