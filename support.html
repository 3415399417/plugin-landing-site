Support

支持与帮助

这里放用户最常遇到的问题。正式上线后，可以把邮箱、飞书表单或反馈链接加到这里。

第一次使用

  1. 安装 Chrome 插件。
  2. 打开插件设置页。
  3. 填写 DeepSeek API Key。
  4. 打开目标网页，点击插件或右键菜单开始使用。

找不到 API Key 设置入口？

点击浏览器工具栏里的插件图标，再点击“设置”；也可以在侧边栏提示中点击“去设置”。

为什么需要 API Key？

当前版本使用你自己的 DeepSeek API Key 发起 AI 请求。这样成本由你自己掌握，插件本身不托管你的 Key。

网页自动提取失败怎么办？

某些网页结构比较特殊，自动提取可能失败。你可以复制正文，打开侧边栏手动粘贴后继续总结或分析；如果小红书正文在图片里，可以上传或粘贴截图，用图文识别提取文字后再分析。

OCR.space API Key 不填可以用吗？

可以用于少量流程测试，但 public test key 不稳定。更建议申请自己的 OCR.space 免费 Key；免费测试时单图建议小于 1MB。

反馈问题时请提供什么？

  • 插件名称和版本号
  • 你使用的网页类型
  • 出错截图或提示文案
  • 你期望得到的结果

反馈入口

正式上线后建议放一个邮箱或表单，例如：support@example.com。当前页面已预留这个位置。
