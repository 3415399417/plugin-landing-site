使用支持

这里整理红势和简阅的上手流程、免费额度、API Key 说明和常见问题。

红势第一次使用

  1. 安装 Chrome 插件。
  2. 打开小红书笔记详情页或搜索结果页。
  3. 点击插件图标、右键菜单，或使用快捷键打开侧边栏。
  4. 粘贴笔记文本，或上传/拖拽/粘贴截图进行 OCR。
  5. 点击“分析爆款结构”，查看可信度、证据引用、复刻建议和改写模板。
  6. 需要创作新内容时，在“带题复刻”中输入自己的选题、产品、经历或观点。

红势是否必须填写 API Key？

不必须。红势默认每天可免费分析 5 次。超过次数后，可以第二天继续使用，或填写自己的 DeepSeek API Key 继续分析。

为什么还保留 API Key 设置？

API Key 适合高频用户、团队测试或希望使用个人额度的人。填写后请求会从浏览器使用个人 Key 发起，不再消耗每日免费额度。

小红书图片文字读取不到怎么办？

红势支持截图 OCR。你可以上传、拖拽或粘贴截图，让插件先识别图片文字，再进入爆款分析。

带题复刻怎么用？

先分析一篇爆款笔记，再输入你的选题、产品、经历或观点。插件会结合爆款结构，输出匹配度、标题、开头、正文结构、完整草稿和风险提醒。

分析结果为什么会提示可信度？

因为 AI 只能基于当前可见文本分析，不知道账号权重、投流、发布时间或真实评论反馈。可信度提示可以帮助你判断结果是否适合参考。

如何删除本地数据？

在插件设置页清空 API Key，清空分析篮，或从 Chrome 扩展管理页删除插件，即可移除本地存储的数据。

简阅说明

简阅用于总结网页、选中文本和粘贴内容。默认每天可免费总结 10 次，也可以填写自己的 DeepSeek API Key 使用个人额度。
