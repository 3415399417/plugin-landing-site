常见问题

先把用户最容易卡住的问题讲清楚。

红势是否必须填写 DeepSeek API Key？

不必须。红势每天免费分析 5 次。超过免费次数后，可以第二天继续使用，也可以填写自己的 DeepSeek API Key。

小红书图片文字读取不到怎么办？

可以使用截图 OCR。上传、拖拽或粘贴截图后，先识别图片文字，再进入爆款分析。

数据会被保存吗？

API Key、每日次数、短期缓存和分析篮保存在本地。免费额度模式下，用户主动提交的文本会发送到官方免费服务并转发给 AI 服务生成结果。

红势会保证笔记爆款吗？

不会。红势是 AI 内容分析助手，帮助复盘结构和辅助创作，不承诺一定涨粉、一定获得流量或商业结果。
