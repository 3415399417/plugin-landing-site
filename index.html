常见问题

上架前先把用户最容易卡住的问题讲清楚。

是否必须填写 DeepSeek API Key？

不必须。简阅每天免费 10 次，红势每天免费 5 次。API Key 是可选高级设置，适合超过免费额度或希望使用个人额度的用户。

自动提取失败怎么办？

可以复制正文后打开侧边栏手动粘贴分析；如果正文在图片里，可以用截图 OCR 识别后再分析。

数据会被保存吗？

API Key、每日次数、短期缓存和分析篮保存在本地。免费额度模式下，你主动提交的文本会发送到官方免费服务并转发给 DeepSeek 生成结果。

后续会收费吗？

当前以免费额度和可选自备 API Key 为主。后续如果增加会员版，会先保留基础免费能力，再增加高频、批量和高级创作功能。
