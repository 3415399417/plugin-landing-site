常见问题

上架前先把用户最容易卡住的问题讲清楚。

是否必须填写 DeepSeek API Key？

简阅可以直接使用每天 10 次免费额度，API Key 是可选高级设置。红势目前仍需要你在设置页填写 DeepSeek API Key。

自动提取失败怎么办？

可以复制正文后打开侧边栏手动粘贴分析；如果正文在图片里，可以用截图 OCR 识别后再分析。

数据会被保存吗？

插件会在本地保存 API Key、短期缓存和分析篮。不会上传到自有服务器。

什么时候可以安装？

Chrome 商店上线后，这里的按钮会替换成正式安装链接。
